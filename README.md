### git 提交规范

```
git commit -m ":bug: fix: 修复问题"

此处的 :bug: 是Gitmoji上对应的提交icon
fix 是提交的类型
```

#### 提交类型 type

- feat: 新功能
- fix: 修复问题
- docs: 修改文档
- ci: 修复 CI 构建问题
- styles: 修改代码格式, 不影响代码逻辑
- refactor: 重构代码
- perf: 提升性能
- revert: 恢复代码
- test: 增加修改测试用例
- wip: 工作进行中
- chore: 修改工具相关(包括不限于文档、代码生成)

[Gitmoji 参考手册](https://gitmoji.dev/)
